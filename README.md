# Практична робота 1

Вебзастосунок з дисципліни **Програмування вебзастосунків** для перерахунку складу палива на суху та горючу масу і визначення нижчої теплоти згоряння.

## Програмна реалізація

- застосунок написаний мовою Go;
- обробка запитів виконується через `net/http`;
- HTML-сторінка формується сервером через `html/template`;
- реалізовано контрольний приклад і варіанти 0-9 з завдання.

## Результат

Застосунок обчислює коефіцієнти `kRS` і `kRG`, показує `Qri`, `Qd`, `Qdaf` та формує таблиці для сухої і горючої маси.

## Висновок

Серверна реалізація на Go дає простий і зрозумілий інструмент для перевірки формул із завдання без використання JavaScript.
